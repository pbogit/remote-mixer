# Behringer XR18

Also works for

* Behringer X18

## Configuration

```js
{
  device: {
    type: 'behringer-xr18',
    options: {
      remoteAddress: '192.168.2.7', // IP address here
      remotePort: 10024
    }
  }
}
```
